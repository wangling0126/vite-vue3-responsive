
# 项目启动
```bash
npm install
npm run dev
```
# 项目说明
🎪 该项目使用技术栈为 vite + vue3 + typescript + vue-router + pinia
🕶 使用pinia代替vuex
⚡ 该项目是响应式的，根据所在设备不同，构建不同的路由表
